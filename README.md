Este projeto consiste no desenvolvimento de um catálogo de séries com foco em design visual e experiência do usuário. A interface foi inspirada em plataformas de streaming, buscando um visual moderno, limpo e atrativo.

O projeto foi construído utilizando HTML e CSS, explorando recursos como animações CSS, efeitos de hover, carrossel automático, tipografia personalizada e organização semântica das seções.

O objetivo principal foi praticar a estruturação de páginas, o estilização avançada com CSS e a criação de interações visuais sem o uso de JavaScript, priorizando simplicidade e aprendizado
